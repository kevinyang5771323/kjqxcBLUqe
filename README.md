# 基于深度学习的人脸识别与管理系统 python589

## 项目概述

本项目是基于深度学习技术实现的人脸识别与管理系统。系统利用Python编写，界面友好，操作简便。通过集成的识别、录入与管理功能，为用户提供了一个高效、准确的人脸识别解决方案。

## 技术栈

- **开发语言：** Python 3.8
- **深度学习框架：** 使用ResNet模型进行人脸识别

## 功能模块

### 人脸识别

- 通过选择人脸图片、视频、摄像头进行已录入人脸的实时识别。
  - 选择人脸照片识别
  - 选择人脸视频识别
  - 摄像头检测人脸识别

### 人脸录入

- 支持通过图片和摄像头检测人脸并录入新的人脸信息。

### 人脸管理

- 系统提供管理和更新人脸数据等功能。

## 系统角色
